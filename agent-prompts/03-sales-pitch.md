# 💼 Sales Pitch with Custom Background

> Copy this entire block and paste it into Claude / OpenDesign / Cursor.

## Agent Instructions

You are building a Hyperframes sales pitch. It features:
1. **Custom AI Background** — A stunning 16:9 image generated by Ideogram.
2. **AI Presenter** — An avatar delivering the pitch on top of the background.
3. **Voiceover** — Premium narration for emotional impact.

## Step-by-Step Execution

### 1. Check existing assets
```bash
node scripts/list_local_assets.js
```

### 2. Generate a custom background image
```bash
node scripts/generate_ideogram.js "sleek futuristic office with holographic displays, 16:9 widescreen, cinematic lighting, photorealistic" background.png
```

### 3. Generate the sales pitch avatar
```bash
node scripts/generate_avatar.js "Imagine cutting your video production time by 90 percent. AssetFlow does exactly that." pitch.mp4
```

### 4. Measure pitch duration
```bash
node scripts/get_media_duration.js pitch.mp4
```

### 5. Generate premium voiceover (ElevenLabs for emotional depth)
```bash
node scripts/generate_elevenlabs.js "Stop wasting hours on manual video editing. Let AI handle the heavy lifting." 21m00Tcm4TlvDq8ikWAM voiceover.mp3
```

### 6. Measure voiceover duration
```bash
node scripts/get_media_duration.js voiceover.mp3
```

### 7. Write the Hyperframes HTML

Use a layered composition: background image as the base, avatar video on top, audio playing alongside.

```html
<!-- BACKGROUND LAYER -->
<div data-start="0" data-duration="PITCH_DURATION">
  <img src="assets/background.png" style="width:100%; height:100%; object-fit:cover;">
</div>

<!-- AVATAR LAYER (overlay) -->
<div data-start="0" data-duration="PITCH_DURATION">
  <video src="assets/pitch.mp4" autoplay muted style="position:absolute; bottom:0; left:0; width:40%; border-radius:12px;"></video>
</div>

<!-- AUDIO LAYER -->
<div data-start="0" data-duration="AUDIO_DURATION">
  <audio src="assets/voiceover.mp3" autoplay></audio>
</div>
```

Replace `PITCH_DURATION` and `AUDIO_DURATION` with exact float values.

## Output Requirements
- Save to `compositions/sales-pitch.html`.
- Use CSS to position the avatar video as an overlay on the background image.
- If audio and video durations differ, loop the shorter one or trim to match.
